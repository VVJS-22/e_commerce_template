# Logo Setup Instructions

## Adding Your Logo

Please save the logo.png file (the square image with 4 quarters) to:

```
/home/jayesh-16632/ecom/frontend/public/assets/images/logo.png
```

The logo should be a square image with 4 quarters containing:
- Dark mode logo
- Light mode logo  
- Round icon
- General logo

## Current Logo Path

The Header component is configured to use: `/assets/images/logo.png`

Once placed, the logo will automatically appear in the header.

## Fallback

If the logo is not present, the header will show a text placeholder temporarily.
